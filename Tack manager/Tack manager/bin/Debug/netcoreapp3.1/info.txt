Привет!
Это программа для управления задачами, эта памятка поможет тебе в работе с программой.
Работа с проектами:
1. Чтобы создать проект тебе нужно вызвать контекстное нажав левой кнопкой мыши в области проектов(Так как у каждой области свое контекстное меню).
2. Чтобы удалить проект тебе нужно выбрать его и снова вызвать контекстное меню.
3. Чтобы переименовать проделай все тоже самое. Просто внизу появляется среда для записи нового имени и кнопка для проверки этого имени.
Работа с пользователями:
1. В поле с пользователями(не исполнителями) можно создавать и удалять пользователей с помощью контекстного меню.
2. Все созданные пользователи будут отражены в рабочей среде для пользователей, и в списке для назначения исполнителей.
Работа с задачами:
1. Для создания задачи ты должен выбрать проект в области проектов, затем под областью для задач, выбрать тип проекта, а дальше вызвать контекстное меню.
(P.S. Выбирать объекты обязательно иначе будут вызываться MessageBox с сообщениями).
2. Для удаления задачи ты должен выбрать задачу, вызвать меню и удалить.
3. Для изменения статуса ты должен выбрать задачу для которой хочешь изменить статус. 
Далее в поле, где отображается статус, выбрать отличный статус, от установленного, а дальше вызвать контекстное меню на выбранной задаче и нажать на Изменить статус.
4. Для назначения исполнителя нужно выбрать задачу, далее в списке под средой для Исполнителей выбрать нужного пользователя, и вызвав контекстное меню присвоить ему задачу.
5. Удаление происходит тоже вызовом контекстного меню и обязательно нужно выбрать задачу.
6. Для изменения пользователя у задач типа Task и Bug нужно удалить пользователя и назначить другого. 
Для задач типа Story стоит ограничение в 3 пользователя для задачи, поэтому чтобы изменить пользователя нужно также удалить и присвоить нового.
7. При удалении пользователя из основного списка, он удаляется и как исполнитель у всех задач.
Работа с подзадачами:
1. Для создания подзадач в поле для задач выбрать задачу типа Epic, дальше в списке для создания подзадач выбрать тип подзадачи, и вызвав контекстное меню,
кликнув на рабочую среду подзадач, создать новую подзадачу.
2. Для подзадач также доступно назначение пользователей и изменение статуса. Изменение статуса для основной задачи влечет изменение статуса для существующей задачи
(Новая подзадача будет иметь статус по умолчанию.)
Ограничения:
1. Ограничение задач:11 штук
2. Ограничение пользователей для Task:1
                                  Bug:1
                                Story:3
